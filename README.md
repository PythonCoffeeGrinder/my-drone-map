# Portfolio Drone Map Orthomosaic

This repository contains a high-resolution, georeferenced orthomosaic map created from aerial imagery captured via drone. It serves as a portfolio piece demonstrating aerial mapping capabilities, precise image stitching, and GIS data readiness.

## Project Details
- **Type:** Drone Orthomosaic (Orthoimage)
- **Application:** Portfolio Display / GIS Web Mapping
- **Formats Included:** GeoTIFF / Optimized PNG

## Features
- High-resolution spatial imagery
- Georeferenced for direct integration into GIS platforms (QGIS, ArcGIS) or web maps (Leaflet, OpenLayers)
- Seamlessly stitched and orthorectified to remove perspective distortion
